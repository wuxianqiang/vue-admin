# 管理后台（组件库）

### 安装依赖
```
yarn install
```

### 启动项目
```
yarn run serve
```

### 在线预览
See [项目地址](https://wuxianqiang.github.io/vue-admin/).

### 包含组件

1. 折叠面板组件
2. 消息提示组件
3. 表单组件（单选按钮，多选按钮，开关按钮，下拉选择框，输入框）
4. 日历选择组件
5. 表格组件
6. 弹窗组件

### 主要功能

1. 登录验证
2. 路由权限
3. 递归组件
4. 面包屑导航
5. 复制、打印

> 所有组件的功能不依赖第三方库实现，如果您需要某个组件单独复制到你的项目就可以直接使用了。
